# Guia rápido de atualização

Este site foi preparado para ser mantido sem mexer no layout.

## Onde editar

Abra o arquivo [conteudo.js](conteudo.js).

Quase tudo que muda com frequência está ali:
- título e chamadas principais
- data do evento
- cidade e local
- link de inscrição
- Instagram e YouTube
- programação por dia
- palestrantes
- perguntas frequentes

## Regra simples

Edite apenas os textos entre aspas.

## Exemplo

Se quiser trocar o tema da edição, procure por:

O evangelho eterno

E substitua pelo novo tema.

## O que não precisa mexer

Não é necessário alterar o HTML nem o CSS para mudanças normais de conteúdo.

## Quando chamar alguém técnico

Só vale pedir ajuda se quiser:
- mudar o visual do site
- criar novas seções
- adicionar integrações diferentes
- mudar animações ou comportamento especial
